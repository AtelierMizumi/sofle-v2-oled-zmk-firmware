# Sofle V2 Quick Reference Card

## Layer Activation
- **SYM**: Hold left thumb (4th thumb key)
- **NAV**: Hold right thumb (2nd thumb key from right)
- **ADJUST**: Hold both SYM + NAV together

---

## Home Row Mods (Hold for 200ms)

### Left Hand
```
A = Ctrl
S = Alt
D = GUI (Win/Cmd)
F = Shift
```

### Right Hand
```
J = Shift
K = GUI (Win/Cmd)
L = Alt
; = Ctrl
```

---

## Quick Access Keys

### BASE Layer
- **Combo ESC**: `U + I` together
- **Tap-Dance `/\`**: Single tap = `/`, Double tap = `\`
- **Delete**: Top-right corner
- **Backspace**: Right inner thumb
- **Enter**: Right outer thumb

### SYM Layer (Hold left thumb)
```
Top Row:    F1  F2  F3  F4  F5  F6  |  F7  F8  F9  F10  F11  F12
Row 2:      ~   !   @   #   $   %   |  ^   &   *   (    )    |
Home Row:   1   2   3   4   5       |  6   7   8   9    0    _
Bottom:     =   -   +   {   }       |  [   ]   <   >    ?
```

### NAV Layer (Hold right thumb)
```
Top Row:    BT_CLR  BT0  BT1  BT2  BT3  BT4
Row 2:      INS  PSCRN  CMENU              |  PgUp  Home  ↑   End
Home Row:   Ctrl  Alt   GUI   Shift        |  PgDn   ←    ↓    →   Del  Bspc
Bottom:     Undo  Cut   Copy  Paste
```

---

## Encoder Functions
- **Left Encoder**: Volume Up/Down (press = Mute)
- **Right Encoder**: Page Up/Down (press = Play/Pause)

---

## Bluetooth Profiles
1. Hold **NAV** layer (right thumb)
2. Top row keys select profiles:
   - Position 1: BT_CLR (clear current)
   - Positions 2-6: BT profiles 0-4

---

## Common Shortcuts

### Programming
- **Curly braces**: SYM + bottom row left (`{` and `}`)
- **Square brackets**: SYM + bottom row right (`[` and `]`)
- **Parentheses**: SYM + row 2 right (`(` and `)`)
- **Underscore**: SYM + home row right pinky (`_`)

### Navigation
- **Vim arrows**: NAV + HJKL
- **Word select**: NAV + Shift (home) + arrows
- **Undo/Redo**: NAV + Z / NAV + Shift+Z

### Clipboard (works on Linux/Windows)
- **Undo**: NAV + Z
- **Cut**: NAV + X
- **Copy**: NAV + C
- **Paste**: NAV + V

## Tips for Learning

### Day 1-3
✓ Focus on BASE layer typing  
✓ Practice home row mods (hold F for Shift)  
✓ Use U+I combo for ESC  

### Week 1
✓ Hold SYM thumb to type numbers  
✓ Practice F-keys on SYM layer  
✓ Get comfortable with symbol placements  

### Week 2
✓ Use NAV + HJKL for navigation  
✓ Practice clipboard shortcuts  
✓ Switch between Bluetooth devices  

### Week 3+
✓ Use modifiers + nav (Shift+arrows for selection)  
✓ Create custom combos for your workflow  
✓ Adjust timing if mods trigger accidentally
